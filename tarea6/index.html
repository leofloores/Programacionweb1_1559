Bienvenidos! Hoy estaremos practicando algunos conceptos y elementos mas importantes vistos hasta el momento en la materia de pw1, ademas veremos el uso de etiquetas semanticas en la estructuracion de un sitio web

Conceptos principales de HTML

HTML

Lenguaje de marcado de hipertexto, que se utilizaba para hacer o estructurar una pagina web

Elemento HTML

Es la unidad basica de estructura y contenido de una pagina web, esta compuesta de:

  1. Etiqueta de apertura
  2. Pares Atributo-Valor
  3. Contenido
  4. Etiquetas de cierre

Etiquetas <head>

Es un etiqueta que contiene metadatos, es la forma en que una pagina web se comunica con el navegador, funciona para el SEO, se pueden enlazar archivos CSS 6 scripts.

Etiqueta<body>

contiene el contenido HTML

Contenedores semanticos

<header> Etiqueta utilizada para el contenido introductorio de una pagina WEB, puede contener el logo el menu de navegacion principal, el nombre de la empresa, etc...
<nav> Etiqueta utilizada para el menu de navegacion principal suele contener enlaces a otras paginas relacionadas con navegar por la pagina principal, no debe haber mas en la pagina
<main> Etiqueta unica de una pagina web cuyo proposito es resaltar nuestro objetivo del mensaje principal del sitio web, dentro se colocara el contenido principal de la pagina web
<section> Es una etiqueta que poco a poco se intenta que sustituya a <div> su uso es dividir la pagina en secciones y agrupar contenido relacionado
<article> Es una etiqueta que se utiliza para agrupar contenido autonomo que si se separa de otro contenido sigue teniendo coherencia normalmente se compone de un encabezado y contenido
<footer> Es una etiqueta que normalmente se encuentra hasta el final de una pagina y funciona como cierre del sitio web a menudo contiene datos de contacto , derechos de autor u otra imformacion complementaria
