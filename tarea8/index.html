Colores multinavegador

Hay mas de 160 colores que son compatibles con todos los navegadores modernos. Estos nombres se pueden usar directamente en css para aplicar colores a elementos sin especificar valores hexadecimales, rgb o hsl

Por ejemplo:

  1. Negro
  2. Blanco
  3. Salmon
  4. Azul
  5. Verde

Colores de un borde

Para que a elemento HTML se le pueda aplicar color a su borde son necesarias 2 propiedades: border-style y border-color

Degradado de colores

Degradado Lineal

El degradado lineal, es una transicion de un color a otro, de forma lineal

Degradado Radial

El degradado radial, es una transicion de un color a otro, de forma radial
